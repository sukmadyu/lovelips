Sambil menunggu waktu libur, asah logika pikirnya dengan belajar coding

Coding adalah kegiatan menulis dan membaca kode untuk dipahami komputer. Dengan belajar coding, seorang coder mampu membuat komputer mengikuti perintahnya.

PT. Coding Indonesia percaya kemahiran anak bermain computer games adalah awal adaptasi anak dengan dunia digital. Semakin sering berlatih coding akan mengasah daya pikir anak untuk menyelesaikan masalah dengan matematis, logis dan analitis. Melalui praktik membuat games, aplikasi dan website, PT. Coding Indonesia mengajak putra-putri Anda mengarungi masa depan menjadi kreator.

About Us

PT. Coding Indonesia berdiri tahun 2013 menyusul kegembiraan anak-anak belajar coding pada program liburan.

Pada 2015, PT. Coding Indonesia mengelola tiga lokasi: Cipete, Kemang dan Bintaro, serta ekstrakurikuler di sekolah-sekolah swasta di DKI Jakarta. Lokasi berikutnya adalah Kelapa Gading dan Depok.
